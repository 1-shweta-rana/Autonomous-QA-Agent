# Product Specifications

## Cart Information
- Items have name, price, and Add to Cart button.
- User can modify quantities in the cart summary.

## Discount Code Rules
- Valid discount code: SAVE15 → applies a 15% discount.
- Invalid or expired codes show: “Invalid discount code!”.
- Discount applies only if subtotal ≥ 100.

## Shipping Rules
- Standard Shipping: Free
- Express Shipping: ₹200

## Payment Rules
- Payment methods: Credit Card, PayPal.
- On successful submission → show “Payment Successful!”.
